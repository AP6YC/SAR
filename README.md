# SAR

Spiking adaptive resonance.

## Installation

### Windows

python=3.9

```shell
conda install pytorch torchvision torchaudio pytorch-cuda=11.7 -c pytorch -c nvidia
pip install snntorch jupyterlab
```

### Linux/Foundry

```shell
pip3 install torch torchvision torchaudio snntorch jupyterlab
```
